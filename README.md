# Excel Handler

Приложение для обработки Excel-файлов с графическим интерфейсом и интеграцией по API.

## 📌 Описание

**Excel Handler** - это десктопное приложение на Python, предназначенное для автоматизации работы с Excel-файлами.

Основные возможности:

* обработка и редактирование Excel-таблиц
* генерация и нормализация данных через API
* удобный GUI-интерфейс на базе Qt
* история операций и логирование

Проект подходит для:

* автоматизации заполнения таблиц
* обработки текстовых данных
* работы с AI внутри Excel-пайплайна

## 📥 Установка

1. Клонируй репозиторий:

```bash
git clone https://github.com/GiveEnd/Excel-handler.git
cd Excel-handler
```

2. Установи зависимости:

```bash
pip install -r requirements.txt
```

---

## ▶️ Запуск

```bash
python main.py
```

## 🤖 Работа с GigaChat

В проекте есть несколько модулей для взаимодействия с API:

* `gigachat_api_question.py` — работа с вопросами
* `gigachat_api_normalize.py` — нормализация текста
* `gigachat_api_promt_*` — генерация и обработка промптов
